# PhotoApp

> A simple app for managing your Photos

## Development

**Clone the repository**

```sh
$ git@github.com:petrbrzek/photoapp.git
```

**Install deps**

```sh
$ cd photoapp
$ yarn install
```

**Launch dev mode**

```sh
$ yarn dev
```

**Build the app**

```sh
$ yarn package
```
